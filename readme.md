# PortSwigger Lab – Blind SQL Injection (Error-Based)

Resolución automatizada en Python de un laboratorio de
**PortSwigger Web Security Academy**
Este script es una automatización de una Blind SQL Injection de tipo Error‑Based, implementada mediante errores condicionales.
##  Técnica
- Blind SQL Injection (error-based)
- Inferencia por errores HTTP 500
- Extracción carácter por carácter
- Automatización con Python

##  Entorno
- Laboratorio educativo de PortSwigger
- Aplicaciones vulnerables controladas
